# Agents Trail logo assets

The logo is based on the selected **Time Gate** concept:

- `agent-trail-mark-primary.svg` — full brand mark for headers and large surfaces
- `agent-trail-mark-compact.svg` — simplified mark for favicon and small UI sizes
- `agent-trail-mark-outline.svg` — low-visual-weight variant for secondary surfaces
- `agent-trail-social-card.svg` — editable source for the Open Graph image
- `agent-trail-logo-time-gate-concept.png` — original generated concept reference

The production SVG marks use the project amber (`#ffb000`) and are designed on a
64×64 grid. The React component uses `currentColor`, so application surfaces
inherit the semantic `text-accent` token in both themes.
