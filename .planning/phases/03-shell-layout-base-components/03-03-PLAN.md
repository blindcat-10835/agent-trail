  ```typescript
  import { cn } from '@/lib/utils'

  interface HudCardProps {
    children: React.ReactNode
    variant?: 'sm' | 'md' | 'lg'
    glow?: boolean
    className?: string
  }

  export function HudCard({ children, variant = 'md', glow = false, className }: HudCardProps) {
    return (
      <div
        className={cn(
          'bg-bg-panel border border-border outline outline-1 outline-offset-[-1px]',
          {
            'hud-clip-sm': variant === 'sm',
            'hud-clip-md': variant === 'md',
            'hud-clip-lg': variant === 'lg',
            'hud-glow': glow,
          },
          className
        )}
      >
        {children}
      </div>
    )
  }
  ```

  **关键点**:
  - 使用 `cn()` 工具函数组合 className（来自 `@/lib/utils`）
  - `variant` prop 控制 clip-path 切角大小（`sm` | `md` | `lg`，默认 `md`）
  - `glow` prop 启用霓虹发光效果（默认 `false`）
  - `outline outline-1 outline-offset-[-1px]` 创建细边框线（补偿 clip-path 切角后的视觉平衡）
  - `bg-bg-panel border border-border` 应用背景色和边框色（Phase 2 已定义的 OKLCH tokens）
  - 条件 className 使用对象语法（`{ 'hud-clip-sm': variant === 'sm' }`），cn() 会正确处理
  - `hud-clip-sm/md/lg` 和 `hud-glow` @utility 来自 Plan 03-01

  **不要**在组件内部硬编码样式值，全部通过 props 和 className 控制。
  **不要**添加默认 padding（由使用方决定内容间距）。
  </action>

  创建新文件 `components/hud/glow-effect.tsx`：

  ```typescript
  import { cn } from '@/lib/utils'

  interface GlowEffectProps {
    children: React.ReactNode
    intensity?: 'low' | 'medium' | 'high'
    color?: string
    className?: string
  }

  export function GlowEffect({ children, intensity = 'medium', color, className }: GlowEffectProps) {
    const glowStyle = {
      '--glow-color': color || 'var(--color-accent)',
      '--glow-spread': intensity === 'low' ? '8px' : intensity === 'high' ? '20px' : '12px',
      '--glow-opacity': intensity === 'low' ? '0.08' : intensity === 'high' ? '0.15' : '0.1',
    } as React.CSSProperties

    return (
      <div
        className={cn('relative', className)}
        style={glowStyle}
      >
        <div
          className="absolute inset-0 pointer-events-none"
          style={{
            boxShadow: `0 0 var(--glow-spread) var(--glow-color), 0 0 calc(var(--glow-spread) * 2) var(--glow-color) / var(--glow-opacity)`,
            opacity: 'var(--glow-opacity)',
          }}
        />
        {children}
      </div>
    )
  }
  ```

  **关键点**:
  - 使用 `cn()` 工具函数组合 className
  - `intensity` prop 控制发光强度（`low` | `medium` | `high`，默认 `medium`）
  - `color` prop 自定义发光颜色（CSS 颜色值，默认 `var(--color-accent)`）
  - 使用 CSS 变量（`--glow-color`, `--glow-spread`, `--glow-opacity`）传递配置到内联样式
  - 发光效果通过绝对定位的子元素实现（`absolute inset-0 pointer-events-none`）
  - `box-shadow` 使用两层：内层紧贴发光，外层扩散光晕
  - `pointer-events-none` 确保发光层不阻挡交互

  **注意**: 这个组件是可选的高级封装，大多数场景可以直接使用 `hud-glow` @utility（Plan 03-01）。GlowEffect 提供更细粒度的强度和颜色控制。

  **不要**在不需要自定义发光颜色或强度的场景使用（直接用 `hud-glow` @utility 更简单）。
  **不要**添加 clip-path 效果（与 HudCard 组合使用）。
  </action>
