<task type="auto">
  <name>任务 3: 添加全局 scanline 和 grid 叠加层</name>
  <files>app/globals.css</files>

  <read_first>
  - app/globals.css（现有文件结构）
  - ../ovao-design/dashboard-hud.html（设计稿的 body::before/after 伪元素实现）
  </read_first>

  <action>
  在 `app/globals.css` 的 `@layer base` 块中添加全局 scanline 和 grid 叠加层（与现有的 `body` 样式同级）：

  ```css
  @layer base {
    * {
      @apply border-border outline-ring/50;
    }
    body {
      @apply bg-background text-foreground;
      position: relative;
    }

    /* Grid overlay - 背景网格线 */
    body::before {
      content: "";
      position: fixed;
      inset: 0;
      z-index: 0;
      pointer-events: none;
      background-image:
        linear-gradient(rgba(95, 212, 255, 0.028) 1px, transparent 1px),
        linear-gradient(90deg, rgba(95, 212, 255, 0.028) 1px, transparent 1px);
      background-size: 48px 48px;
    }

    /* Scanline overlay - 扫描线效果 */
    body::after {
      content: "";
      position: fixed;
      inset: 0;
      z-index: 0;
      pointer-events: none;
      background: repeating-linear-gradient(
        0deg,
        transparent,
        transparent 3px,
        rgba(0, 0, 0, 0.06) 3px,
        rgba(0, 0, 0, 0.06) 4px
      );
    }
  }
  ```

  **关键点**:
  - `body::before` 实现 grid 叠加层（交叉网格线，48px 间距，cyan 色 rgba(95, 212, 255, 0.028)）
  - `body::after` 实现 scanline 叠加层（水平扫描线，每 4px 重复，rgba(0,0,0,0.06) 黑色半透明）
  - `pointer-events: none` 确保伪元素不阻挡页面交互
  - `z-index: 0` 确保在内容下方（页面内容默认 z-index: auto）
  - `position: relative` 添加到 body 上，确保伪元素的 fixed 定位正确

  **不要**在 `html` 或其他元素上添加伪元素，只在 `body` 上添加。
  </action>

  <verify>
  <automated>grep -n "body::before\|body::after" app/globals.css</automated>
  </verify>

  <done>
  - `body::before` 存在且实现 grid 叠加层（background-image 使用两个 linear-gradient）
  - `body::after` 存在且实现 scanline 叠加层（repeating-linear-gradient）
  - 两个伪元素都有 `pointer-events: none` 和 `z-index: 0`
  - body 元素有 `position: relative`
  </done>
