## Objective

Create the Activity Console page at `/activity` with Summary cards (Total Events, Errors, Sources breakdown) and a LogBrowser component that displays filtered log entries with search and expandable details. **Output**: A fully functional Activity Console page with summary statistics, filterable log browser, search functionality, and expandable log entry details.
