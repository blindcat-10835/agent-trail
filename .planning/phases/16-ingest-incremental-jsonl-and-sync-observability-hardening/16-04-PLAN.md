</objective>

<threat_model>
T-16-06: Debug endpoints may leak local file contents or expose unbounded memory history. Mitigation: include only metadata, cap recent history with a fixed-size ring buffer, and do not serialize raw JSONL lines or message contents.
T-16-07: Adding parser concurrency can reintroduce the original memory amplification. Mitigation: default to serial parsing and require explicit bounded config for any concurrency or SQLite batching.
</threat_model>
