## Trust Boundaries

| Boundary | Description |
|---|---|
| Untrusted JSONL/tool data -> React render | Markdown and tool input content originates from local agent logs. |
| Replay UI -> clipboard | Copy actions expose only user-selected replay content. |
| Trace file path strings -> UI | Paths shown are trace data only, not used for filesystem reads. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-09-08 | Tampering | `MarkdownContent` highlighting | mitigate | Use `react-markdown` component overrides, keep children string, and avoid `dangerouslySetInnerHTML`. |
| T-09-09 | Information Disclosure | edit formatter path display | mitigate | Display only path strings already present in trace data; do not read files or expand arbitrary paths. |
| T-09-10 | Elevation of Privilege | tool formatter | mitigate | Formatter is pure display code and never executes tool input or shell text. |
| T-09-11 | Denial of Service | diff display | mitigate | Generate bounded string previews from existing input fields; no expensive diff library or unbounded fetch is introduced. |
</threat_model>
