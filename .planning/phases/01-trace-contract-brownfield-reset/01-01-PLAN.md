   - Export `interface TraceSession` with fields: id (string), source (TraceSource), project (string), startedAt (string | null), endedAt (string | null), status (SessionStatus), rootSessionId (string | undefined), parentSessionId (string | undefined), relationshipType ('root' | 'subagent' | 'fork' | 'continuation' | undefined), metrics (SessionMetrics), turns (TraceTurn[])

   - Export `interface TraceTurn` with fields: id (string), sessionId (string), index (number), userMessage (TraceMessage | null), assistantMessages (TraceMessage[]), activities (TraceActivity[]), startedAt (string | null), endedAt (string | null), durationMs (number | null), tokenUsage (TokenUsage | undefined)

   - Export `interface TraceMessage` with fields: id (string), ordinal (number), role ('user' | 'assistant' | 'system' | 'tool_result'), content (string), timestamp (string | undefined), model (string | undefined), tokenUsage (TokenUsage | undefined), sourceMetadata (SourceMetadata)

9. **Activity union**:
   - Export `type TraceActivity = TraceToolCall | TraceSkillUse | TraceSubagentLink | TraceThinkingBlock | TraceSystemEvent`

10. **Tool call**:
    - Export `interface TraceToolCall` with fields: type ('tool_call'), id (string), name (string), category (ToolCategory), inputJson (string), resultEvents (TraceToolResultEvent[]), status ('pending' | 'success' | 'error'), error (string | undefined), durationMs (number | undefined)

    - Export `type ToolCategory = 'Bash' | 'Edit' | 'Read' | 'Grep' | 'Task' | 'Agent' | 'Other'`

12. **Tool result event**:
    - Export `interface TraceToolResultEvent` with fields: type ('result_event'), timestamp (string | undefined), content (string), isPartial (boolean)

13. **Skill use**:
    - Export `interface TraceSkillUse` with fields: type ('skill_use'), name (string), inputSummary (string), result (string | undefined), status ('success' | 'error')

14. **Subagent link**:
    - Export `interface TraceSubagentLink` with fields: type ('subagent_link'), subagentSessionId (string), subagentSource (TraceSource), relationship ('spawned' | 'attached')

15. **Thinking block**:
    - Export `interface TraceThinkingBlock` with fields: type ('thinking'), content (string), isRedacted (boolean)

16. **System event**:
    - Export `interface TraceSystemEvent` with fields: type ('system'), subtype (string), content (string)

17. **Token usage**:
    - Export `interface TokenUsage` with fields: inputTokens (number), outputTokens (number), cacheReadTokens (number | undefined), cacheWriteTokens (number | undefined)

18. **Source metadata**:
    - Export `interface SourceMetadata` with fields: sourceType (TraceSource), sourceFile (string), sourceLine (number | undefined), sourceVersion (string | undefined), cwd (string | undefined), gitBranch (string | undefined)
