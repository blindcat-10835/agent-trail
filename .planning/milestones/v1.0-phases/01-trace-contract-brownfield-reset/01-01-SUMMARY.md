## Key Decisions Made

### D-04: Trace Contract Independence
- **Decision:** No imports from gateway/types.ts in types/trace.ts
- **Rationale:** Multi-source architecture requires canonical model independent of any single source
- **Outcome:** Complete separation achieved, verified by grep and manual review

### D-14: Dual-Status Model
- **Decision:** TraceSourceMetadata has both ingestStatus and optional gatewayStatus
- **Rationale:** Claude/Codex don't have Gateway, but OpenClaw has both file-based and real-time status
- **Outcome:** Interface supports both or only ingestStatus, tests verify flexibility

### Test Framework: Vitest over Jest
- **Decision:** Use vitest instead of Jest
- **Rationale:** Per RESEARCH.md - better ESM/TypeScript support, faster watch mode
- **Outcome:** Successfully configured and running, all tests pass
