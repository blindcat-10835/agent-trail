## Architecture

```
~/.openclaw/workspace-{agentName}/IDENTITY.md  →  parseIdentityMarkdown  →  avatar filename
~/.openclaw/workspace-{agentName}/{filename}    →  resolveAvatar()        →  image bytes + MIME type

GET /api/v1/agents/:name/avatar  →  Hono route  →  Response(image)

Frontend: AgentAvatar → <img src="/api/agent-tools/openclaw/agents/{name}/avatar"> → fallback glyph
```
