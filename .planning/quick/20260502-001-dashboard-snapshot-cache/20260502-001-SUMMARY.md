## What Was Built

### `lib/dashboard-snapshot-cache.ts`
- `SNAPSHOT_KEY = "ovao.dashboard.snapshot.v1"` with version `1` and 5-minute TTL
- `StoredSnapshotV1` serializes `agents` as `[string, AgentInfo][]` entries (Map-safe JSON)
- `DashboardSnapshot` interface exposes typed Map back to consumers
- Module-level `memCache` avoids repeated `JSON.parse` within the same session
- All localStorage access guarded by `typeof window !== "undefined"` for SSR safety
- Corrupted/expired entries are silently cleared and `null` returned

### `stores/gateway/gateway-store.ts` (additive only)
- Added import for `readDashboardSnapshot` / `writeDashboardSnapshot`
- Added `hydrateFromCache: () => void` to `GatewayState` interface
- `writeDashboardSnapshot` call after `set()` in `fetchDashboardData` â€” uses local `rpcAgents` variable directly (no re-read from state)
- `hydrateFromCache` action: reads snapshot, short-circuits if `state.agents.size > 0` to prevent overwriting live data, sets `isDashboardLoading: true` to trigger SYNCING indicator
- Zero existing lines modified

### `components/hud/gateway-bootstrap.tsx`
- `hydrateFromCache()` called synchronously before `init()` in the `useEffect`
- Dependency array updated: `[disconnect, hydrateFromCache, init]`

### `components/hud/shell-status-bar.tsx`
